Habit Tracker

Sleep 7.5–8h /night, same window, dark-cold-quiet room +

Why: Sleep is the firmware update for cognition, emotional regulation, and spinal repair—your back will thank you.

How: Sleep diary + wearable HRV or reaction-time test; if latency >15 min or waking >2×/night, fix environment.

If Not: You stay a high-performing zombie: code full of off-by-one bugs, patience at ∅ with daughter.

Lift heavy (compound) + mobility 3×/wk +

Why: Muscle ≈ “bullet-proof body armor” against back pain; strength = metabolic spare battery.

How: Log lifts; add 2.5 kg or 1 rep every week.

If Not: Creeping dad-bod, chronic lumbar whining, and eventual “why does everything hurt?” existential dread.

90 g+ protein + mostly plants & fish; caffeine ≤400 mg by 2 pm; alcohol <2 drinks/wk +

Why: Fuels recovery, prevents late-night doom-scroll distortions, keeps neurochemistry sane.

How: Track macros 2 weeks each quarter; body-fat calipers or DEXA yearly.

If Not: Roller-coaster energy, gut inflammation, foggy compile-time in brain.

Deep-work blocks (2 × 90 min, phone in Faraday pouch) +

Why: Context switches slaughter throughput; uninterrupted flow is where 80% of code quality lives.

How: Daily log of focused minutes; review pull-request defect rate.

If Not: Lots of busy noise, zero compounding output—AKA career plateaus.

Digital purge: ruthlessly mute, archive, or delete 90% of inputs +

Why: Attention is your scarcest currency; garbage signals breed shallow thinking.

How: Weekly audit: count >10 open browser tabs? You failed.

If Not: Permanent low-grade cognitive load; inability to distinguish signal from hype.

Weekly 1-on-1 with your future self (Roam/Obsidian journal) +

Why: Systems engineer your life: reflect, retro, set next iterations—exactly like production post-mortems.

How: 30-min Sunday review note; track OKR completion % monthly.

If Not: Drift into reactive mode; strategic goals rot in Trello limbo.

Therapy or high-caliber coach bi-weekly +

Why: Even rationalists have blind spots; an external debugger steps through recursive trauma loops.

How: After 3 months you should articulate one “core schema” you’ve unearthed.

If Not: Same emotional edge-cases keep seg-faulting relationships.

Novelty adventures 1×/wk (new park, science kit, bike trail) +

Why: Deposits memories, resets perspective; also stealth cardio.

How: Photo journal; if two consecutive weeks show couch photos—you’re slipping.

If Not: Not a legacy you want.

Monthly “monk mode” weekend: off-grid, hike, or silent retreat +

Why: Pattern-interrupt for strategic clarity; default network resets, new insights surface.

How: Can you state one non-obvious decision Monday morning? If not, retreat was fake.

If Not: Burnout sneaks up; creativity flatlines into incremental refactors only.

Quarterly audit of “S-curve” skills (e.g., Rust, RL-HF, multimodal infra) +

Why: Future employability = stacking S-curves before old ones commoditize.

How: Pick 1 skill/quarter; produce artifact: blog, open-source PR, or talk.

If Not: Wake up obsolete, watching juniors eclipse you with fresh tooling.
